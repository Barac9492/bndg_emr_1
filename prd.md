# [PRD] OSINT 기반 응급의료 자원 최적화 플랫폼: "분당 골든패스"

## 1. 제품 개요 (Product Overview)

- **목적**: 분당 내 주요 거점 병원의 실시간 가용 상태를 OSINT 데이터로 예측하여, 구급대원이 '수용 가능한 병원'을 단 한 번에 찾아 이송 성공률을 극대화함.
- **핵심 가치**: "전화 돌리기 전, 이미 답을 알고 있다."
- **타겟 사용자**: 119 구급대원, 권역응급의료센터 상황실 요원, 지자체 보건 행정관.

---

## 2. 핵심 문제 정의 (Problem Statements)

- **데이터 지연**: 공공 API(E-Gen) 데이터와 실제 현장 상황 사이에 10~20분의 시차가 존재함.
- **보이지 않는 변수**: 병상의 유무보다 '전문의 집도 가능 여부'가 중요하지만, 이는 정적 데이터에 잘 나타나지 않음.
- **예측 부재**: 현재 비어있어도, 5분 뒤 사고 환자가 몰릴 가능성을 고려하지 못함.

---

## 3. 주요 기능 (Key Features)

### 3.1. 하이브리드 실시간 대시보드 (Predictive Dashboard)
- **기능**: 공식 응급의료 API + OSINT(교통 사고, SNS 사고 제보) + 병원 내부 회전율 데이터를 통합한 시각화 지도.
- **상태 지수(Status Index)**: 단순 '여유/포화'가 아닌, 향후 30분 내 수용 가능 확률을 %로 표시.

### 3.2. OSINT 기반 수요 예측 엔진 (Demand Forecaster)
- **사고 감지**: T맵 정체 데이터 및 SNS 키워드(판교역 사고, 분당선 고장 등)를 크롤링하여 응급실 유입 증가를 사전 경고.
- **이벤트 연동**: 성남시 내 대형 행사, 날씨(빙판길 등) 변수를 점수화하여 가중치 부여.

### 3.3. '지능형 이송' 추천 알고리즘 (Smart Triage)
- **환자별 최적 매칭**: KTAS(응급도)와 질환별(외상, 뇌, 심장)로 해당 전문의가 '실제로 집도 가능한' 최단 거리 병원 추천.
- **병목 우회**: 분당서울대병원이 '레드(Full)'일 때, 용인세브란스나 성남시의료원의 실시간 잔여 용량을 비교 제안.

---

## 4. 데이터 아키텍처 (Data Architecture)

| 데이터 레이어 | 수집 소스 (Source) | 역할 |
|---|---|---|
| 정적 데이터 | 공공데이터포털, 병원 홈페이지 | 병원 기본 정보, 의료 장비 보유 현황 |
| 동적 데이터 | 응급의료포털(E-Gen) API | 현재 가용 병상 수, 가동 중인 장비 |
| OSINT 데이터 | X(트위터), 맘카페, 교통 API, 기상청 | 실시간 사고 발생 여부, 시민 제보 상황 |
| 행동 데이터 | 구급차 GPS, 이동 경로 | 병원 앞 구급차 대기 수량 (간접 정체 지표) |

---

## 5. 성공 지표 (Success Metrics - KPIs)

- **재이송률(뺑뺑이 비율) 감소**: 분당 내 응급 환자의 병원 결정 후 수용 거부 발생 건수 50% 감축.
- **병원 선정 시간 단축**: 사고 접수 후 최종 이송지 결정까지 걸리는 시간 3분 이내.
- **예측 정확도(MAPE)**: 향후 1시간 내 응급실 포화 상태 예측 정확도 85% 이상 유지.

---

## 6. 제약 사항 및 리스크 (Constraints & Risks)

- **데이터 신뢰성**: SNS 허위 정보(Fake News)로 인한 오판 가능성 → 교차 검증 알고리즘 필요.
- **개인정보보호**: 환자 식별 정보가 포함되지 않도록 데이터 비식별화 처리 엄격 적용.
- **의료진 수용성**: 시스템이 제안하는 병원과 의료진의 실제 판단이 충돌할 경우의 책임 소재.

---

## 7. 로드맵 (Roadmap)

- **Phase 1 (MVP)**: 분당 내 3개 핵심 병원(서울대, 차, 제생) 대상 실시간 데이터 대시보드 구축.
- **Phase 2 (AI 고도화)**: OSINT 연동 및 사고 발생 시 유입량 예측 모델 배포.
- **Phase 3 (광역 확장)**: 용인, 광주 등 인접 지자체와 데이터 연동 및 구급대원 전용 앱 보급.
